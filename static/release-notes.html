Decentraleyes v1.4.1



Pro Tip: Would you rather not see this page after installations or updates? You can disable it by navigating to about:config (in a new tab), then searching for BoFifL9V, and setting showReleaseNotes to false.


New: Additional languages. Support was added for the languages Arabic, Brazilian Portuguese, Czech, Estonian, Hebrew, Icelandic, Korean, Romanian, Serbian (Cyrillic), and Turkish.

New: The request sanitizer now also erases "Cookie" and "Origin" headers. Said details will be automatically stripped from any allowed requests to supported Content Delivery Networks.

New: Additional mappings for Chinese users. Added support for the Chinese "BootCDN" delivery network, and popular Chinese CDN mirrors "Geekzu Public Service" and "USTC Linux User Group".

Enhanced: Improved support for various prevalent shorthand URL notations.
